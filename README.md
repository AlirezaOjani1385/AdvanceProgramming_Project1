# 🎬 Cinema Reservation System (C++ Console Application)

یک سیستم متنی و تحت کنسول ساده و کارآمد برای مدیریت سالن‌های سینما، اکران فیلم‌ها و رزرو صندلی توسط مشتریان که به زبان **C++** نوشته شده است.

## 📌 ویژگی‌های برنامه
* **مدیریت سالن‌ها:** ظرفیت‌دهی پویا به سالن‌ها (سالن ۱: ۱۰ صندلی، سالن ۲: ۲۰ صندلی، سالن ۳: ۳۰ صندلی).
* **برنامه‌ریزی اکران (Scheduling):** جلوگیری از تداخل زمانی اکران فیلم‌ها در یک سالن مشترک.
* **سیستم رزرو هوشمند:** بررسی خودکار وضعیت صندلی‌ها، جلوگیری از رزرو مجدد صندلی‌های پر شده و کاهش ظرفیت سالن پس از هر رزرو موفق.
* **مرتب‌سازی هوشمند:** 
  * نمایش لیست فیلم‌ها به ترتیب **زمان اکران**.
  * نمایش وضعیت صندلی‌ها به ترتیب **شماره صندلی**.

## 🛠 دستورات برنامه (Commands)

برنامه به صورت مداوم ورودی‌های زیر را از کاربر دریافت و پردازش می‌کند:

### ۱. افزودن سان فیلم (`add_show`)
برای اضافه کردن یک فیلم جدید به برنامه اکران سالن.
* **فرمت دستور:** `add_show <movie_name> <time> <hall_number>`
* **مثال:** `add_show Inception 18 2` (اکران فیلم اینسپشن ساعت ۱۸:۰۰ در سالن ۲)

### ۲. رزرو صندلی (`reserve`)
برای رزرو یک صندلی خاص برای یک مشتری در یک سان مشخص.
* **فرمت دستور:** `reserve <hall_number> <time> <seat_number> <customer_name>`
* **مثال:** `reserve 2 18 5 Ali` (رزرو صندلی شماره ۵ سالن ۲ ساعت ۱۸ برای علی)

### ۳. لیست فیلم‌های یک سالن (`list_movies`)
نمایش تمام فیلم‌های اکران شده در یک سالن خاص به ترتیب زمان.
* **فرمت دستور:** `list_movies <hall_number>`
* **مثال:** `list_movies 2`

### ۴. گزارش وضعیت صندلی‌ها (`report_seats`)
نمایش لیست صندلی‌های رزرو شده و نام مشتریان برای یک سانس خاص.
* **فرمت دستور:** `report_seats <hall_number> <time>`
* **مثال:** `report_seats 2 18`

## 🏗 ساختار داده‌ها (Data Structures)

این پروژه با استفاده از قابلیت‌های مدرن C++ (مانند STL) طراحی شده است:
* **`vector`:** برای ذخیره‌سازی داینامیک لیست فیلم‌ها و مشتریان.
* **`map`:** برای مدیریت ظرفیت اولیه هر سالن.
* **`Lambda Expressions`:** استفاده از توابع لامبدا همراه با `std::sort` و `std::find_if` جهت جستجو و مرتب‌سازی بهینه داده‌ها.
